Countdown Timer
Overview:
The Enhanced Countdown Timer is a web-based application that allows users to set a countdown target, view the remaining time in days, hours, minutes, and seconds, and track progress via a visual progress bar. The application also includes features to pause, resume, and reset the countdown.

Features:
Set Countdown Target: Users can input a target date and time.
Live Countdown Display: Shows days, hours, minutes, and seconds remaining.
Progress Bar: Visual representation of elapsed time relative to the total countdown duration.
Pause and Resume: Allows users to pause and resume the countdown.
Reset: Resets the countdown timer to its initial state.
Technologies Used
HTML: Structure of the web application.
CSS: Styling of the application for a modern and responsive design.
JavaScript: Functionality for the countdown timer, progress bar, and button actions.
